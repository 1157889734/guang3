一、准备工具：网线、电脑
二、更新操作：
    2.1 用网线把触摸屏和电脑连接起来，修改电脑ip地址：192.168.1.60。
    2.2 解压升级文件，这里是G3_LCD_20171017_1729_V2.3，把解压出来的文件放在和tftpd32.exe同一目录下，
打开tftpd32.exe。
    2.3 打开电脑运行界面，在快速启动菜单中，或者直接按win+r 按键直接唤出运行窗口，输入cmd命令按回车。
    2.4 输入 telnet 192.168.1.20 ，等待连接成功， 输入用户 root， 密码为空直接按回车即可。
    2.5 输入 cd /root ,进入 root 目录。
    2.6 输入ps命令查看进程号，输入命令kill -9 xxx 杀死看门狗进程watchdog.sh，同样输入kill -9 xxx杀死
主程序/root/src -qws（xxx代表的是进程编号，如看门狗的编号是862，也可能是别的数字，参照ps打印出来的编号） 
    2.7 输入 tftp -g -r src 192.168.1.60 命令，时间较长等待传输完毕。
    2.8 输入 chmod 777 src ， 至此程序已经更新成功，如果不更新配置，可以直接输入 reboot 命令重启设备，
或者断电重启，然后在查看版本处查看程序是否更新成功。    
    2.9 如果要更新配置文件，在2.8操作完后，不重启，输入 tftp -g -r setTempBroName.ini 192.168.1.60 
命令更新紧急广播的命名。
    
三、配置文件说明：
    3.1 目前程序已经自带了30条紧急广播，最多支持45条，如果有需要修改，可以通过修改配置文件即可，如果
没有变化则可以不用配置。
    3.2 setTempBroName.ini 配置文件是紧急广播命名的文件，通过修改setTempBroName.ini文件的内容可以更新
紧急广播的命名，目前最多支持45条。可以用记事本或者其他工具打开。
    
四、注意事项：
    4.1 如果2.4连接不成功，可以ping 192.168.1.20，查看网络是否连通。
    4.2 如果网络连通而Telnet连接不上，则查看电脑是否打开Telnet服务，打开操作可自行百度。也可使用其他
软件比如"SecureCRT.exe"连接Telnet。
